#  Escrevedeira

Projeto desenvolvido como parte da disciplina de **Programação em Laboratório 2** da **UFCG**, com o objetivo de praticar conceitos de programação, manipulação de arquivos, testes e lógica de código.

**Escrevedeira** é o nome do projeto, inspirado em um termo que remete à escrita e à elaboração. A palavra *escrevedeira* pode significar uma ave ou ainda remeter ao ato de escrever no contexto da linguagem popular, conforme o dicionário Aulete.

---

##  Estrutura do Repositório

```
proglab2_ufcg/
└── Escrevedeira/
    ├── src/
    │   └── escrevedeira---           # Código-fonte principal
    └── teste/
        └── escrevedeira              # Testes para validação do código
 ```

As principais funcionalidades do projeto envolvem:

- Entrada e saída de dados
- Manipulação de strings e arquivos
- Uso de estruturas condicionais e de repetição
- Aplicação de estruturas de dados básicas
- Verificação do comportamento do sistema por meio de **testes automatizados com JUnit**

O projeto pode ser adaptado conforme o enunciado do laboratório, podendo representar, por exemplo, um sistema simples, um jogo ou um processador de texto.

---

##  Como Executar

### Pré-requisitos

Antes de executar o projeto, é necessário ter instalado:

 **Java JDK** (versão 11 ou superior)  
 **Eclipse IDE**  
 **JUnit** configurado no projeto (geralmente já incluso nos projetos de laboratório)

---

### Importando o Projeto no Eclipse

1. Abra o **Eclipse**
2. Clique em **File → Import**
3. Selecione **Existing Projects into Workspace**
4. Escolha a pasta raiz do repositório

### teste/escrevedeira

2. Clique com o botão direito sobre a classe de teste
3. Selecione **Run As → JUnit Test**

Os testes automatizados validam se as funcionalidades implementadas estão de acordo com o comportamento esperado pelo enunciado do laboratório.
