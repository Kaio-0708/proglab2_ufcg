# ProgLab2 - UFCG

Repositório destinado aos estudos e implementações das disciplinas **Programação 2** e **Laboratório de Programação 2** da **Universidade Federal de Campina Grande (UFCG)**.

Contém exercícios, projetos práticos e anotações desenvolvidos ao longo do curso, com foco em:

- Orientação a objetos (POO)
- Boas práticas de programação
- Uso da linguagem **Java**
- Estruturas de dados e algoritmos básicos

## Objetivo do Repositório

Este repositório tem como objetivo:

- Consolidar o aprendizado das disciplinas **Programação 2** e **Laboratório de Programação 2**.
- Servir como referência pessoal para implementação de conceitos de **Java**.
- Organizar exercícios e projetos práticos em um único local.
