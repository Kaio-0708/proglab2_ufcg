# Jogo do Bicho 

##  Objetivo do Aprendizado
Este projeto foi desenvolvido com o objetivo de **aplicar e consolidar os conceitos fundamentais da linguagem Java**, incluindo:

- Estrutura básica de um programa Java (`main`, métodos e classes);  
- Manipulação de **arrays** e **loops**;  
- Uso da classe **`Scanner`** para leitura de dados do usuário;  
- Utilização da classe **`Random`** para geração de números aleatórios;  
- Prática de **métodos estáticos** e **organização modular do código**;  
- Implementação de **condicionais (`if` / `else`)** e **estruturas de repetição (`while`, `for`)**;  
- Aprendizado de boas práticas de programação e legibilidade de código.  

---

##  Descrição do Projeto
O **Jogo do Bicho** é uma versão simplificada em Java do clássico jogo popular.  
O programa permite que o jogador:
1. Escolha **5 números de 1 a 25** correspondentes aos “bichos”;  
2. Veja sua **cartela organizada** com os nomes dos bichos escolhidos;  
3. O sistema **sorteia aleatoriamente** um número entre 1 e 25;  
4. O programa informa **se o jogador ganhou** (acertou o número sorteado) ou **não ganhou**.  

---

##  Funcionalidades
-  Cadastro de aposta com 5 números válidos (1 a 25);  
-  Exibição dos bichos correspondentes a cada número;  
-  Sorteio aleatório de um número (representando o bicho vencedor);  
-  Verificação automática se o jogador acertou o número sorteado;  
-  Feedback textual indicando o resultado da aposta.

---
